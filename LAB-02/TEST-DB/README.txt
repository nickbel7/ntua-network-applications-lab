# Έλεγχος επικοινωνίας Java Εφαρμογών με μία Σχεσιακή Βάση.
# =======================================================================

# 1. Δημιουργία Σχεσιακής Βάσης
# -----------------------------------------------------------------------

  Αρχικά δημιουργήστε την Σχεσιακή Βάση ακολουθώντας τις οδηγίες που 
  υπάρχουν στον φάκελο «SQL».


# 2. "Τρέξιμο" JAVA κώδικα εφαρμογών
# -----------------------------------------------------------------------

  Έπειτα δημιουργήστε και «τρέξτε» τα προγράμματα που υπάρχουν στον 
  φάκελο «JAVA» με την ακόλουθη σειρά. 

  (i) Αρχικά «τρέξτε» το πρόγραμμα που υπάρχει στον φάκελο «TestDB». 
  Στόχος του προγράμματος αυτού είναι να βεβαιωθείτε ότι μπορείτε να 
  επικοινωνήσετε με την βάση από μία Desktop Εφαρμογή. 

  (ii) Έπειτα «τρέξτε» το πρόγραμμα που υπάρχει στον φάκελο «ServletTestDB». 
  Στόχος του προγράμματος αυτού είναι να βεβαιωθείτε ότι μπορείτε να 
  επικοινωνήσετε με την βάση από μία Web Εφαρμογή που βρίσκεται σε έναν 
  εξυπηρετητή (στην δική μας περίπτωση τον Apache Tomcat). 

# * Σημείωσεις:
# -----------------------------------------------------------------------   

  Απαραίτητη προϋπόθεση για την επικοινωνία με μία σχεσιακή βάση είναι 
  να βρίσκεται ο DB-Connector (DB-specific JAR αρχείο) στο «Classpath» 
  της εφαρμογής. 

  Για την Web Εφαρμογή αυτό μπορεί  να γίνει με πολλούς τρόπους. Στην 
  περίπτωσή μας έχουμε τοποθετήσει το JAR αρχείο στον φάκελο 
  «WEB-INF/Lib» για να το «βρει» η εφαρμογή μας όταν τρέχει.
