# 1. Δημιουργία Σχεσιακής Βάσης
# -----------------------------------------------------------------------

  Βεβαιωθείτε ότι το XAMPP Control Panel «τρέχει» και η 
  MySQL βάση καθώς και ο Apache (PHP) Server έχουν ξεκινήσει.

  Δημιουργήστε μια βάση με όνομα "LAB2DB" και ακολούθως κάντε 
  import τους πίνακες & τα δεδομένα που υπάρχουν στο αρχείο "lab2db.sql".
   
   - DB Name: LAB2DB
   - DB "content": SQL/lab2db.sql

# 2. "Τρέξιμο" JAVA κώδικα εφαρμογών
# -----------------------------------------------------------------------
	
  Κάντε "import" το Project στο Eclipse και ακολούθως «Τρέξτε»:
  
   - Αρχικά την κλάση: ProgramDB
   - Έπειτα την κλάση: ProgramWindowMain

# ** Προσοχή **
# -----------------------------------------------------------------------

  Θα χρειαστεί να κάνετε update το «classpath»
  
  --> Project Properties --> Java Build Path --> Classpath -->
       - Remove Existing Jar since the path is not correct
       - Add External Jar (i.e., the exact path to the DB Connector JAR File)
